# Cài đặt Ubuntu
** Cài đặt process và memory cho hệ điều hành**
![enter image description here](https://lh3.googleusercontent.com/sceVALbg94ucnTL-jLM5Mrgjb2Pszd8Lh9llLQtnVWtVFkmzMG8FaLVMITJWdH7bL3Vtg6FDjho)
** Tiến hành cài đặt trên hệ điều hành Ubuntu**
- Chọn ** " Install Ubuntu Server" **

![enter image description here](https://lh3.googleusercontent.com/mUBLeZ9c64FbFVxJkjL1m9tRVxiQKaW2MIgLxRKhHeIVVqmQE2p1LjfVDvwSDjF14HYIPqfSXCw)
- Tiếp theo chọn phần ** " Select a Language" ** , phải chọn ngôn ngữ cho hệ điều hành 

![enter image description here](https://lh3.googleusercontent.com/iB96XKrC2ip_OeswDpDqLtF_BhTOa3RBHc0MrSk86Z1F9Ka2yDRMpy_6XRJuo8UMcuSfqJtukXo)
- Bước tiếp theo lựa chọn khu vực của bạn

![enter image description here](https://lh3.googleusercontent.com/WsQvhScVoie_8NHHoHgPxkcEDx7049MzP1hzmo-fWhPn7tHOWjTmvI7F-uxTs07QwLxCCzb57ys)

- Bước tiếp theo lựa chọn ngôn ngữ cho keyboard
![enter image description here](https://lh3.googleusercontent.com/WbKmh_-u3w_EPKPq76E4b5Ur0ZW2dNlhcJVjISvHAHRtG8ZxHVIg7iLdVdxuldnSng_CIImwBdU)
- Tiếp tục cài đặt :
![enter image description here](https://lh3.googleusercontent.com/TdvsDWHcK1mMks7SipEDaPVT5eysxG_yEX4WLZXgjwyRRdJppSUyC6x9lKKlodWpjOBz596Xzg4)
- Lúc này bạn sẽ cấu hình tên dùng đại diện (host name) cho server Ubuntu của bạn khi hiển thị hệ điều hành 

![enter image description here](https://lh3.googleusercontent.com/PoYMRKfG8t1F9xo_pIQWngAvt1lDm6eo3BGwTD80SZvlvnJXM1PbqT52S34sbikTxf6hmetoPcU)
- Bước tiếp theo bạn sẽ tạo 1 tài khoản user khác để đăng nhập lại từ đầu. Thay vì sử dụng user quyền cao nhất là **root** vì lý do bảo mật 
![enter image description here](https://lh3.googleusercontent.com/B8mQCHOk9mWdTrSui1pQKA67GT0NHnO0EC-voaZFgqOrbGDD68lBMf-JLXz2xWvGENubjGxqrdY)
- Chọn timezone của Ubuntu Server, lúc này timezone sẽ tự chọn múi giờ phù hợp
![enter image description here](https://lh3.googleusercontent.com/3u8OnXB0sFSQF1y3TR3uE5Do-6wRF-4MuDK1fUIVj6N9uMxeY8Jip3LnnpwQRM4PpGv4bF41XkM)
- Lựa chọn cấu hình thông qua các partition ổ cứng 
![enter image description here](https://lh3.googleusercontent.com/u6djbGkUwRCn9Fd_M7Py9XMkM4lLbOpljGmwjF1ILmHzDEe6-Lr2E47wb4YOyAxB-oHl8AZpjsw)

![enter image description here](https://lh3.googleusercontent.com/qiEz1vJklShdGbjIGDxb0LIoH8udD6fZkiK5w14O1Yfj3PyaZP0izh8HaU-ZEwgsHSx-K89V39k)

- Xác nhận chép đè thông tin partition lên ổ cứng đã lựa chọn trước đó . Nếu trước đó OS server đã có thông tin phân vùng khác thì sẽ bị mất và không thế khôi phục lại. Chọn Yes.
![enter image description here](https://lh3.googleusercontent.com/4oy7EdEGADNbxEdyuE2tpTv0j4JC3BpS02g3SvgXS_SGxDsGEz2l5g3bF-QAcO5mdQFR_klYNLw)

- Cài đặt các chương trình chuẩn bị sẵn cho server Ubuntu
![enter image description here](https://lh3.googleusercontent.com/kE6kT6B_uh10l5wCfSCVlV_1iBzF44WeTaJZjmkXRfDf_Rc6Knpsu6zp3ONFKPfgPvDk3soqLv4)
- Hoàn thành quá trình cài đặt, để hệ thống tự chạy
